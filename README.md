# Linux Infrastructure Journey

## Mission

This repository documents my journey of learning Linux and Operating Systems from first principles.

The goal is not merely to learn commands, but to understand how modern infrastructure systems work underneath:

* Operating Systems
* Processes and Scheduling
* Memory Management
* Filesystems
* Networking
* Cybersecurity Infrastructure
* Cloud Infrastructure
* AI Infrastructure
* Future Space Infrastructure Systems

---

## Learning Philosophy

I believe infrastructure engineers should understand what happens beneath the abstraction layer.

Instead of memorizing commands, I focus on:

* Systems Thinking
* Architecture Understanding
* Internal Mechanisms
* Real-world Infrastructure Applications
* Hands-on Experiments
